# 📊 Power BI Portfolio

Dieses Repository dokumentiert meine Projekte im Bereich Datenanalyse und Datenvisualisierung mit Microsoft Power BI.
Ziel ist es, Daten strukturiert aufzubereiten, zu analysieren und in verständlichen Dashboards darzustellen.

---

## 🧠 Inhalte & Kompetenzen

* Datenbereinigung und -aufbereitung (Power Query)
* Datenmodellierung (Beziehungen, Star Schema)
* Erstellung interaktiver Dashboards
* DAX-Grundlagen (z. B. Measures, KPIs)
* Visualisierung von Geschäftskennzahlen

---

## 📁 Projekte

### 1. Sales Dashboard

**Beschreibung:**
Analyse von Verkaufsdaten mit Fokus auf Umsatz, Produktkategorien und Zeitverlauf.

**Inhalte:**

* Umsatzentwicklung (monatlich)
* Top-Produkte
* Regionale Auswertung

**Dateien:**

* `sales_dashboard.pbix`
* `sales_data.csv`
